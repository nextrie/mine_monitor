# MineMonitor
A Minecraft server Monitor

## Installation

Installation requires node package manager (npm)

```bash
git clone https://github.com/nextrie/mine_monitor
```
```bash
npm install
```

## Usage

```bash
cd mine_monitor
npm start
```
or
```bash
cd mine_monitor
electron .
```
